# 🧪 Laboratorio de Datos

Bienvenido a este espacio donde los datos dejan de ser números aburridos  
y empiezan a contar historias.

---

## 📊 ¿Que trabajo realizaremos en esta materia?

En **Laboratorio de Datos** vamos a explorar, analizar y modelar información utilizando herramientas de ciencia de datos.  
El objetivo es transformar datos crudos en conocimiento útil, aplicando técnicas modernas de análisis y aprendizaje automático.

---

## 🐧 Dataset de trabajo: Pingüinos

A lo largo de la cursada trabajaremos con el famoso dataset de **pingüinos**, que incluye información sobre distintas especies, tales como:

- 🧬 Especie
- 📏 Longitud del pico
- ⚖️ Masa corporal
- 🏝️ Isla de origen
- 🚻 Sexo

Este dataset nos permitirá:

- Explorar datos reales
- Detectar patrones
- Aplicar modelos de clasificación
- Visualizar información de manera efectiva

---

## ⚙️ Herramientas

Durante el curso utilizaremos:

- 🐍 Python
- 📚 Librerías de análisis de datos (pandas, matplotlib, seaborn)
- 🤖 Técnicas de Machine Learning
